# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Hassan-src/Todo-list.git
```

Go to the project directory

```bash
cd Todo-list
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
